# 작성 규칙

* 문서 생성은 마크다운 기본문서로 생성한다. (.md 확장자)
* 언어나 기술명으로 폴더를 만들고 root에 문서를 생성하지 않는다.
* 파일명과 commit message 모두 영어로 한다.
* 마크다운 문서에 수식이 포함될 경우 pdf 파일도 같이 commit한다.
* commit을 위한 commit을 하지 말고 매일 배움을 통해서 commit한다.
